Step-by-step Walkthrough

What happens in a real session

01
Visitor arrives at heritage site

The robot is stationed at the site entrance. The visitor approaches and scans the RFID tag provided at reception. The robot powers on and runs a 3-second self-check.

02
Handle extends if user is detected

The touch sensor on the handle detects if a user places their hand near it. The servo automatically extends the telescopic handle to full length. The robot announces the site name and language selection prompt via the speaker.

03
Visitor selects language and tour begins

One press of the language selector cycles through available languages. The robot confirms the choice and begins narrating the site introduction. Movement starts smoothly, with the robot leading the visitor along the recommended path.

04
Narration updates as visitor moves

As the robot approaches each monument or section, GPS and camera data trigger new narration blocks. The visitor hears specific facts about each structure as they stand in front of it.

05
Tour ends — robot returns to standby

At the end of the defined route, the robot announces the tour is complete, retracts the handle, and returns to a low-power standby state. The visitor receives a farewell message in their chosen language.
